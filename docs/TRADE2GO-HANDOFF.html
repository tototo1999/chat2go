3 · 模型 / 路由 本次变更

当前:经 OpenRouter,模型 Claude Sonnet 4.6

  • worker _anthropic_client():env 有 OPENROUTER_API_KEY → Anthropic(api_key=or_key, base_url="https://openrouter.ai/api")(OpenRouter 的 Anthropic-skin,原生 Messages 协议,tool use 透传)。
  • DEFAULT_MODEL:OpenRouter 模式 = anthropic/claude-sonnet-4.6(点号);直连模式 = claude-sonnet-4-6(横杠)。
  • 可逆开关:删 Modal secret 里的 OPENROUTER_API_KEY + redeploy → 自动切回直连官方 Anthropic。
  • 每次请求会打一行日志:[chat2go] room=… provider=openrouter model=anthropic/claude-sonnet-4.6 industry=外贸跟单 trade=True。

验证(2026-05-31):OpenRouter 直测 200 + tool_use 透传正常;线上 3 次真实请求 200 OK(8–12s);OpenRouter 用量从 0 起计。
