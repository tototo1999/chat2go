2. 软硬件必选项 (Must-Haves) 与延展 (Extensions) 矩阵

我们明确区分了实现“真身音容复刻与亲子编程启蒙”底座的**绝对必选项（Must-Haves）**，以及保障未来不断进化的**能力延展图谱（Extensions）**：

必选项 MUST-HAVE

🧠 ESP32-S3 SoC (8MB PSRAM)

主控芯片必选项。双核 LX7 架构提供强悍音视频流软解算力，8MB 超大 PSRAM 保证 MJPEG/H.264 家庭 Vlog 视频和音频缓存不卡顿。

必选项 MUST-HAVE

🔊 MAX98357A I2S DAC 音频系统

发声系统必选项。通过高性能 I2S 数字音频接口，完美高保真还原亲人通过 Verbal Coding 克隆出的温暖、清爽的真实嗓音，拒绝普通蜂鸣器的廉价感。

必选项 MUST-HAVE

🎤 MEMS 数字麦克风 (I2S 总线)

收音系统必选项。提供高信噪比高清音频录入，为孩子的日常口语对话、古诗背诵以及家人的声音极速克隆提供无可比拟的底层声学底座。

必选项 MUST-HAVE

🧲 N52 环形钕铁硼强磁与充电底座

物理挂载必选项。环形 N52 超强钕铁硼提供高达 18N 的拉力，稳吸在冰箱门等铁表面，配合 Pogo-Pin 充电底座提供随放随充、无摩擦的温馨体验。

必选项 MUST-HAVE

🌐 Web Audio & WebSocket 实时双向网络

软件通信必选项。Web Audio 保证浏览器端音波捕获，WebSocket 保证跨设备磁贴之间毫秒级信息同步与物理微振动推送。

必选项 MUST-HAVE

📷 Claude Vision API 视觉答疑

多模态软件必选项。将孩子微距拍摄的练习册难题精准识别，让代表爸爸妈妈探索之眼的 Agent 给出鼓励式解题引导。

延展项 EXTENSION

📖 家庭记忆 RAG 向量知识库

核心能力延展。大咖上传资料及家庭账单日记向量化（pgvector），让 Agent 真正“记住”家人的重要纪念日、生活喜好和温暖点滴。

延展项 EXTENSION

📡 BLE / Wi-Fi Mesh 局域自组网

网络拓扑延展。当家中路由器断网时，多款冰箱贴之间能自动通过 BLE 蓝牙或 Wi-Fi 局域进行直连通信，保障紧急呼救与便签传送永不断线。

延展项 EXTENSION

🖨️ 温情 Zine 纸质排版实体打印联动

输出模态延展。支持月底将家庭一月来的照片和暖心便签一键排版，通过低成本热敏打印机直接打印出属于家人的实体黑白 Zine 迷你小杂志。

延展项 EXTENSION

👓 AR 眼镜与可穿戴生态打通

多终端生态延展。支持向雷鸟 X2、Rokid 等 AR 眼镜和智能手环推送紧急通知或日程，打破冰箱贴的地理限制。
