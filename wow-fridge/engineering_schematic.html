1. 物理硬件架构系统框图

Voilà AI 冰箱贴电路基于集成度极高、低功耗的 ESP32-S3 片上系统，主控通过高速 SPI 和 I2C/I2S 接口连接液晶显示屏、MEMS 数字麦克风、数模转换音频放大器及电源管理模块。

2. Dandelion OTA 无线传送与编译管线

当开发者在 PC 终端输入 wow deploy 时，系统会自动对代码进行预编译压缩，并将高清多媒体资源（视频/图像）重采样适配为 480x480 分辨率后通过无线 OTA 一键瞬间部署到物理磁贴上。

sequenceDiagram autonumber actor Dev as 💻 开发者电脑 (PC) participant CLI as 🚀 wow CLI 终端 participant Cloud as ☁️ Vercel / 云端编译服务器 participant Device as 🏡 物理冰箱贴 (ESP32-S3) Dev->>CLI: 输入 `wow deploy ./my-app` CLI->>CLI: 打包用户 Python 脚本及多媒体静态资源 CLI->>Cloud: 发送部署包 (POST /api/deploy) Note over Cloud: 云端资源优化与代码静态校验 Cloud->>Cloud: 优化/压缩 PNG 图像和 MP4 视频为 480x480 尺寸 Cloud->>Cloud: 将 Python 脚本编译为 MicroPython 字节码 (.mpy) Cloud-->>CLI: 返回编译校验成功 (Compilation OK) Note over Device: 处于 Wi-Fi 低功耗监听休眠态 (Listening) Cloud->>Device: 通过 WebSocket 强行唤醒并触发 OTA 部署 Device->>Device: 液晶屏切换为“正在传送... 🚀”Clay 装载动画 Cloud->>Device: HTTPS 安全下载字节码与重采样媒体资源 Note over Device: 将编译好的字节码和静态媒体资源写入 SPIFFS 闪存 Device->>Device: 软重启 MicroPython 虚拟机并执行 `main.mpy` Device->>Device: 液晶屏重载为成功运行 App 的“小黄人/全家福/视频”状态 Device-->>Cloud: 响应部署确认 (Acknowledge OK) Cloud-->>CLI: 传送成功响应 CLI-->>Dev: 控制台打印 "🎉 传送成功！磁贴已重载！" 庆祝状态

3. Python SDK 低层硬件映射关系

Voilà AI 极简 Python 接口设计高度抽象，屏蔽了微控制器底层驱动寄存器的繁琐配置，将硬件交互抽象为最温暖的家庭感知指令：

极简 Python SDK 接口 底层物理通道 / GPIO 口 数据总线接口 底层芯片驱动程序 物理功能与情境应用
Magnet.screen.show("Hello") GPIO 10, 11, 13, 14, 15 SPI Bus MicroPython Framebuffer + LVGL 在玻璃显示屏上呈现字样、小图标或交互画布
Magnet.screen.play_video("vlog.mp4") PSRAM DMA 缓存控制 DMA Stream MJPEG / H.264 Soft Decoder 在宝贝和妈妈磁贴上完美渲染流畅的家庭 Vlog 循环视频
Magnet.mic.listen() GPIO 4 (BCLK), 5 (WS), 6 (DATA) I2S Input I2S DMA Audio Reader 通过高灵敏度 MEMS 麦克风录入家人的语音留言
Magnet.speaker.say("沙拉做好啦") GPIO 1 (BCLK), 2 (LRCK), 3 (DATA) I2S Output MAX98357A DAC + TTS Stream 通过微型喇叭合成并播放甜美悦耳的系统合成家庭人声
Magnet.sensors.detect_motion() GPIO 8 (SDA), 9 (SCL), 7 (INT) I2C Bus MPU6050 Driver (6-Axis) 感知冰箱门开启、磁吸掉落或双击震动并触发事件
Magnet.status.set_glow("#ff007f") GPIO 12 (氛围灯信号线) Single-Wire ESP32 RMT Controller (WS2812B) 在磁铁外壳底盘外发散出一圈温暖柔和的梦幻数码呼吸光影
